Ce dossier contient les exercices effectués dans le cadre du cours OpenClassroom "Reprenez le contrôle à l'aide de Linux"

Pics contient un script permettant de générer un fichier html affichant les miniatures des images contenues dans le dossier spécifié

Dico contient un script permettant de compter le nombre d'occurences d'une lettre dans un dictionnaire passé en paramètre, ou le nombre de mots commençant par chaque lettre.
